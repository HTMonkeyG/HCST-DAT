# HCST-DAT
## 概述
&emsp;&emsp;高速指令同步技术（High Speed Command Synchronize Tech.），原指对于MCBE指令微时序的一系列基于实验的研究。后引出一特殊分支HCST设计辅助理论（Design Assisting Theory），尝试系统地将指令设计过程进行规范化与理论化。\
&emsp;&emsp;本仓库系列文章将介绍笔者对于HCST-DAT的研究成果。
